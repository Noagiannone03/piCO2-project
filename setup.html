🚀 Configuration de votre Pico

Guide étape par étape pour configurer votre capteur My Pico et commencer à surveiller la qualité de l'air

📋 Avant de commencer

Ce tutoriel vous guide pour configurer votre capteur My Pico en quelques minutes. Assurez-vous d'avoir votre Pico sous tension et prêt pour la configuration.

1

Préparation du Pico

Préparez votre capteur My Pico pour la première configuration.

⚠️
Important !
Ne débranchez jamais le Pico pendant la configuration. Cela pourrait corrompre les données et nécessiter une réinitialisation.
  • a
    Brancher le Pico
    Connectez votre Pico à l'alimentation USB et attendez qu'il s'allume
  • b
    Vérifier l'écran
    L'écran doit afficher le mode configuration avec les instructions de connexion
  • c
    Noter l'ID du device
    Notez l'ID affiché sur l'écran (ex: picoAZ12) - vous en aurez besoin plus tard
2

Connexion au réseau WiFi du Pico

Connectez-vous au point d'accès WiFi créé par votre Pico pour la configuration.

  • a
    Ouvrir les paramètres WiFi
    Sur votre smartphone ou ordinateur, ouvrez les paramètres WiFi
  • b
    Rechercher le réseau
    Cherchez un réseau nommé "My-Pico-[ID]" (ex: My-Pico-picoAZ12)
  • c
    Se connecter
    Connectez-vous avec le mot de passe: mypico123
💡
Astuce
Si le réseau n'apparaît pas, redémarrez votre Pico en le débranchant 10 secondes puis en le rebranchant.
3

Configuration du WiFi

Configurez la connexion WiFi de votre domicile sur le Pico.

  • a
    Ouvrir la page de configuration
    Une page devrait s'ouvrir automatiquement. Sinon, allez sur 192.168.4.1 dans votre navigateur
  • b
    Sélectionner votre réseau
    Choisissez votre réseau WiFi domestique dans la liste
  • c
    Entrer le mot de passe
    Saisissez le mot de passe de votre WiFi et cliquez sur "Configurer"
⚠️
Attention
Assurez-vous que votre réseau WiFi fonctionne correctement et que le signal est suffisant où se trouve le Pico.
4

Enregistrement automatique

Une fois connecté au WiFi, votre Pico s'enregistre automatiquement dans le système.

  • a
    Connexion WiFi
    Le Pico se connecte à votre réseau WiFi (LED/écran confirme la connexion)
  • b
    Enregistrement Firebase
    Le Pico s'enregistre automatiquement dans la base de données avec sa géolocalisation
  • c
    Début des mesures
    Après préchauffage du capteur (30 secondes), les mesures CO2 commencent automatiquement
💡
Info
La géolocalisation est approximative et basée sur votre adresse IP. Vous pourrez l'affiner plus tard dans les paramètres.
